# 🏦 Bank Management System

A simple Bank Management System built using Python and SQLite.  
It allows users to create accounts, login, and perform banking operations.

## 🚀 Features
- Create new account
- Secure login with PIN
- Credit (deposit money)
- Debit (withdraw money)
- Check account balance
- View transaction history

## 🛠️ Technologies Used
- Python
- SQLite (Database)

## ▶️ Run
python bank_system.py

## 📌 How It Works
- Stores account details in SQLite database
- Maintains transaction history
- Uses functions for different banking operations
- Menu-driven system for user interaction

## 📁 Files
- bank_system.py → Main program
- bank.db → Database file (auto created)
